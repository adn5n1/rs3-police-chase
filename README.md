# RS3 Police Chase

A 2D police chase game developed in Python using Tkinter for my BTEC Level 3 IT Games Development unit.

## About

RS3 Police Chase is a simple arcade-style game where the player controls an Audi RS3 while collecting cash bags and avoiding police cars. The aim is to achieve the highest score possible before running out of lives or time.

## Features

- Main menu
- Instructions screen
- Difficulty selection (Easy, Medium and Hard)
- Score system
- Lives system
- 60 second timer
- Police AI that follows the player
- Cash bag collection
- Game Over screen
- Play Again and Main Menu options

## Controls

| Key | Action |
|------|--------|
| ↑ ↓ ← → | Move the RS3 |
| R | Restart game (Game Over screen) |
| M | Return to Main Menu (Game Over screen) |
| Esc | Exit the game |

## Built With

- Python 3
- Tkinter
- Visual Studio Code
